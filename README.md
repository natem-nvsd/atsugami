Atsugami
========
Tool for managing and tagging your images.

# Dependencies  
PostgreSQL (Tested on 14 and 14.1)  
GTK3  
C11-compliant C complier (Tested with Clang 11.0.1 have)

# Building  
`make`  


# Installing  
` sudo make install`
